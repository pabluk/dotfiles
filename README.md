# My dotfiles #


## tmux configuration ##


A simple tmux configuration file


### Usage ###


Download `.tmux.conf` to your home directory

```bash
$ cd ~
$ wget https://github.com/pabluk/dotfiles/raw/master/tmux/.tmux.conf
```
